# Python_Diwali_Sales_Analysis
# 🪔 Diwali Sales Data Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to understand customer purchasing behavior, identify high-value customers, and uncover sales trends.

The goal is to help businesses make better marketing and sales decisions during festive seasons.

---

## 📊 Key Objectives
- Analyze customer demographics (age, gender, marital status)
- Identify top-performing states and occupations
- Find high-revenue product categories
- Understand purchasing patterns during Diwali

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains information about:
- Customer details
- Product categories
- Purchase amounts
- Location data

---

## 🔍 Key Insights
- Married women aged 26–35 are the top buyers
- Highest sales come from states like Uttar Pradesh, Maharashtra, and Karnataka
- IT, Healthcare, and Aviation professionals spend more
- Food, Clothing, and Electronics are top-selling categories
